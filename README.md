# 暗号資産詐欺チェッカー

ホワイトペーパーを分析して暗号資産詐欺のリスクを評価するWebツール

## 特徴

- 🔍 **AI駆動の分析**: Google Gemini 2.0 Flash APIを使用した高度な分析
- 📊 **詳細なリスク評価**: 0-100のスコアと3段階のリスクレベル
- 📥 **結果のエクスポート**: PDF・CSV形式でダウンロード可能
- 📜 **分析履歴の保存**: localStorage活用で最大20件まで自動保存
- 🔄 **複数比較機能**: 最大3つのホワイトペーパーを並べて比較
- ⚠️ **包括的な免責事項**: 法的助言ではないことを明示
- 🎨 **モダンなUI**: アニメーション効果とレスポンシブデザイン
- 💯 **完全無料**: Vercel無料枠で運用可能
- 🚀 **APIキー不要**: ユーザーはAPIキーを取得する必要なし
- ⚡ **高速読み込み**: CDNの事前接続とdeferによる最適化

## アーキテクチャ

- **フロントエンド**: HTML/CSS/JavaScript（静的ファイル）
- **バックエンド**: Vercelサーバーレス関数（Node.js）
- **AI API**: Google Gemini API（サーバーサイドで管理）

## 使い方

### エンドユーザー向け（デプロイ済みサイト）

1. デプロイされたサイトにアクセス
2. 免責事項を読んで同意
3. ホワイトペーパーを入力（3つの方法）:
   - テキスト直接貼り付け
   - URL入力（※一部制限あり）
   - PDFアップロード（※開発中）
4. 「詐欺リスクを分析」ボタンをクリック
5. 結果を確認

## ローカルでの開発・テスト

### 必要なもの

- Node.js（v18以上推奨）
- Google Gemini APIキー

### セットアップ手順

```bash
# 1. Vercel CLIをインストール
npm install -g vercel

# 2. プロジェクトディレクトリに移動
cd path/to/crypto-scam-checker

# 3. 環境変数ファイルを作成
# .env.exampleをコピーして.envにリネーム
cp .env.example .env

# 4. .envファイルにAPIキーを設定
# GEMINI_API_KEY=your_actual_api_key

# 5. ローカル開発サーバーを起動
vercel dev

# 6. ブラウザでhttp://localhost:3000にアクセス
```

### HTMLのみのプレビュー（API機能なし）

```bash
# Python 3を使用
python -m http.server 8000

# ブラウザでhttp://localhost:8000にアクセス
# 注意: API機能は動作しません
```
